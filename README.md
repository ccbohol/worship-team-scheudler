# Worship Team Scheduler

Static worship-team scheduling app built with plain HTML, CSS, and JavaScript.

## Local use

Open `index.html` in a browser.

## Push to GitHub

1. Create an empty GitHub repository.
2. In this folder, run:

```bash
git add .
git commit -m "Initial worship team scheduler"
git remote add origin <your-repo-url>
git push -u origin main
```

## Deploy

### GitHub Pages

1. Create a GitHub repository and push this project to the `main` branch.
2. In GitHub, open `Settings > Pages`.
3. Set the source to `GitHub Actions`.
4. Push to `main` again if needed, then wait for the `Deploy Static Site to GitHub Pages` workflow to finish.

### Netlify

1. Create a new site from this folder or from the GitHub repository.
2. Use the publish directory `.`.
3. No build command is required.

### Vercel

1. Import this folder or GitHub repository into Vercel.
2. Keep it as a static site.
3. No build command is required.
